# Software Prerequisites

The following software packages need to be installed in this directory:

* **Jackal**: [https://honig.c2b2.columbia.edu/jackal](https://honig.c2b2.columbia.edu/jackal)
* **SPIDER2**: [https://zhouyq-lab.szbl.ac.cn/download/](https://zhouyq-lab.szbl.ac.cn/download/)
* **MIBPB5**: [https://weilab.math.msu.edu/MIBPB/](https://weilab.math.msu.edu/MIBPB/)
* **MS_Intersection**: [https://weilab.math.msu.edu/MIBPB/](https://weilab.math.msu.edu/MIBPB/)
